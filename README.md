## ✓ 프로젝트 소개

**만다라트**는 `1개의 핵심 목표와 8개의 세부 목표, 64개의 실천 과제를 작성함으로써 목표를 설정하고 이뤄내기 위해 계획을 세울 수 있는 도구` 입니다.

온라인에서 쉽게 작성하고 이미지로 출력할 수 있도록 하여 접근성과 편의성을 높였습니다.

<br />

## ✓ 주요 기능

- 만다라트 작성
- 자동 저장 기능
- 이미지 출력 기능
